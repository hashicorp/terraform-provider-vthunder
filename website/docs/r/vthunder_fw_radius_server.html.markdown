# vthunder\_fw\_radius\_server

`vthunder_fw_radius_server` Provides details about vthunder fw radius server
## Example Usage


```hcl
provider "vthunder" {
  address  = "129.213.82.65"
  username = "admin"
  password = "admin"
}

resource "vthunder_fw_radius_server" "FwRadiusTest" {
	listen_port = "1024" 
}
```
