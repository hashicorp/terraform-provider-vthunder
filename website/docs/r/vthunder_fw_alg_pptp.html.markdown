# vthunder\_fw\_alg\_pptp

`vthunder_fw_alg_pptp` Provides details about vthunder fw alg pptp
## Example Usage


```hcl
provider "vthunder" {
  address  = "129.213.82.65"
  username = "admin"
  password = "admin"
}

resource "vthunder_fw_alg_pptp" "FwAlgTest" {
	default_port_disable = "default-port-disable" 
}
```
