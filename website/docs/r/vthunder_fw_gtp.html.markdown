# vthunder\_fw\_gtp

`vthunder_fw_gtp` Provides details about vthunder fw gtp
## Example Usage


```hcl
provider "vthunder" {
  address  = "129.213.82.65"
  username = "admin"
  password = "admin"
}

resource "vthunder_fw_gtp" "FwTest" {
	gtp_value = "enable" 
}
```
